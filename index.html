SOBRE

Curso de introdução ao Bootstrap oferecido pela Digital Innovation One - DIO
Bootstrap é um framework com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web, usando HTML, CSS e JavaScript.
Possui vasta biblioteca de componentes, melhorando a experiência do usuário em um site amigável e responsivo.

CONCLUSÃO

Bootstrap é um framework que tem ganhado cada vez mais popularidade entre os desenvolvedores front-end.
Pode ser usado facilmente, poupando tempo dos desenvolvedores ao programar.

É muito flexível e pode acomodar quase todas as necessidade do desenvolvimento web front-end.
Suas funções permitem que páginas na internet funcionem em telas de qualquer tamanho, desenvolvimento responsivo.
